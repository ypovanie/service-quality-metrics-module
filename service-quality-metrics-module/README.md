# Модуль расчёта показателей качества обслуживания

Данный модуль разработан в рамках курсовой работы по теме:

**«Проектирование и разработка информационной системы учёта и анализа показателей качества обслуживания»**

## Краткое описание

Модуль предназначен для расчёта основных показателей качества обслуживания на основе данных по обращениям клиентов и результатам обратной связи.

С его помощью можно получить сводные значения показателей, которые в дальнейшем могут использоваться в информационной системе для анализа качества обслуживания.

## Реализованные функции

В модуле реализованы:

- расчёт показателя удовлетворённости клиентов **CSAT**;
- расчёт показателя лояльности клиентов **NPS**;
- расчёт среднего времени обработки обращения **AHT**;
- расчёт показателя решения вопроса с первого обращения **FCR**;
- расчёт соблюдения регламентного времени обслуживания **SLA compliance**;
- формирование сводной информации для просмотра результатов;
- тестирование основных функций модуля.

## Структура проекта

```text
app/
  __init__.py                 # создание Flask-приложения
  services/
    metric_service.py         # бизнес-логика расчёта метрик
  routes/
    dashboard.py              # HTTP-маршрут со сводной аналитикой

tests/
  conftest.py                 # фикстуры для тестов
  test_metric_service.py      # unit-тесты расчётов
  test_dashboard_routes.py    # тест HTTP-маршрута

requirements.txt              # зависимости проекта
README.md                     # описание проекта и инструкция по запуску
.gitignore                    # исключения для Git
```

## Используемые технологии

- Python 3.11+
- Flask
- pytest
- pytest-cov

## Запуск

```powershell
python -m venv .venv
.\.venv\Scripts\Activate.ps1
pip install -r requirements.txt
python -m flask --app app:create_app run
```

После запуска откройте в браузере:

```text
http://127.0.0.1:5000/dashboard/summary
```

## Запуск тестов

```bash
pytest -v
```

Для расчёта покрытия кода:

```bash
pytest --cov=app.services.metric_service --cov-report=term-missing
```

## Пример результата

Маршрут `/dashboard/summary` возвращает JSON со сводными значениями метрик качества обслуживания, например:

```json
{
  "aht": 117.5,
  "csat": 4.67,
  "fcr": 100.0,
  "nps": 67,
  "requests_total": 2,
  "sla_compliance": 100.0
}
```
